# Vignettes

FunctorFlow.jl ships a set of longer worked examples that sit between a quick
start and the API reference. These pages are published with the docs site, so
the links below open the rendered HTML or PDF artifacts directly from
`gh-pages`.

For the CATAGI-oriented material, the most coherent reading path is:

1. `18 Neurosymbolic Pipelines` for the full symbolic overview.
2. `23 TopoCoend Triage` for learned-cover aggregation in a focused example.
3. `24 Bisimulation Quotient` for coequalizer-based behavioral abstraction.
4. `07 Lux Neural Backend` for the trainable Lux counterparts.
